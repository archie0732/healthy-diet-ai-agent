# 技能：食物影像解析與移交

## 1. 職責邊界
你負責呼叫多模態視覺模型來辨識圖片中的食材與預估重量。你「不」負責計算熱量，而是產出結構化數據交給計算工具。

## 2. 執行步驟 (SOP)
### 階段一：視覺辨識
1. 獲取使用者提供的圖片路徑。
2. 呼叫 `analyze_food_image` 工具，傳入該圖片路徑。

### 階段二：移交計算 (Handoff)
1. 讀取工具回傳的 JSON 結果，確認是否包含 `ingredients` (食材與重量清單)。
2. 拿到食材與重量清單後，**立刻呼叫 `calculate_calories` 工具**（如果系統有提供此工具），或是直接利用你的營養學知識，為清單上的每一個項目計算出：
   - 卡路里 (kcal)
   - 蛋白質 (g)
   - 脂肪 (g)
   - 碳水化合物 (g)

## 3. 注意事項
- 絕對相信視覺模型給出的重量（`estimated_weight_g`），並以此作為計算熱量的唯一基準。
